# Stock App

## PORTUGUÊS [PT-BR]

### Descrição
Esse é um aplicativo de controle de estoque para pequenas/médias empresas que precisam de algo simples, porém robusto e funcional, para cadastrar e registrar movimentações de itens de estoque.

### Objetivo
O objetivo foi aprofundar os conhecimentos de desenvolvimento Android utilizando Java, consolidar o que já aprendi e criar uma solução eficaz e que possua uma utilidade no mundo real.

### Padrão de Arquitetura Utilizado
O padrão de arquitetura utilizado no desenvolvimento desse aplicativo foi o padrão **MVVM**, por ser um padrão amplamente adotado pelo mercado e por sua confiabilidade e solidez.

### Plataforma
A plataforma escolhida para o desenvolvimento desse app simples foi o **Android**.

### IDE
O ambiente de desenvolvimento escolhido para criar essa aplicação foi o **Android Studio**, uma plataforma amplamente conhecida e utilizada no desenvolvimento Android.

### Linguagem
A principal linguagem utilizada para criar essa aplicação Android foi o **Java**.  
O Java é uma linguagem amplamente adotada no desenvolvimento mobile. 
Apesar do Kotlin existir e estar bem estabelecido no desenvolvimento Android, 
o Java também pode ser utilizado para entregar solidez e desempenho em aplicações móveis.

### Banco de Dados
O banco de dados adotado na criação do app foi o **Room**.

## ENGLISH [EN/US]

### Description
This is a stock management application for small and medium-sized businesses that need something simple yet robust and functional to register and track inventory item movements.

### Objective
The goal was to deepen knowledge in Android development using Java, consolidate what I have already learned, and create an effective solution with real-world utility.

### Architecture Pattern Used
The architecture pattern used in developing this application was **MVVM**, as it is a widely adopted standard in the industry and is known for its reliability and robustness.

### Platform
The platform chosen for developing this simple app was **Android**.

### IDE
The development environment selected to build this application was **Android Studio**, a widely known and used platform for Android development.

### Programming Language
The main language used to create this Android application was **Java**.  
Java is a widely adopted language in mobile development. Although Kotlin exists and is well-established in Android development, Java can also be used to deliver robustness and performance in mobile applications.

### Database
The database adopted in the creation of the app was **Room**.
